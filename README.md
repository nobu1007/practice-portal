# Practice Portal

チーム学習・訓練管理のための統合ポータルサイト。

## 機能

- **タスク管理 + EXP/レベル** — 訓練タスクの進捗管理と管理者承認フロー
- **クイズシステム** — レベル別問題と忘却曲線による復習スケジューリング
- **マインドマップ** — ノードベースの知識管理ツール
- **管理画面** — 招待コード管理・メンバー進捗確認

## セットアップ

セットアップ手順は [supabase/README.md](supabase/README.md) を参照してください。

- HTML / CSS / JavaScript（バニラ）
